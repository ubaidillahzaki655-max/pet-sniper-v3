# 🎯 Pet Sniper v3.0

Auto Pet Sniper untuk game **Grow A Garden** di Roblox dengan fitur auto server hop dan multi-pet search.

## 📋 Fitur

✅ **Multi-Pet Search** - Cari 3 pet sekaligus (Spring bee, Jerboa, Nyala)
✅ **Auto Server Hop** - Otomatis pindah server jika tidak ada pet
✅ **Auto Click** - Otomatis membeli pet saat ditemukan
✅ **Sound Alert** - Notifikasi suara saat pet ditemukan
✅ **Real-time Stats** - Monitor progress secara real-time
✅ **Customizable** - Atur harga, timing, dan pet target
